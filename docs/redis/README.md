# redis 简介

**Redis 是一个基于内存的 key- value 数据库，也可用作缓存和消息代理。**

它支持多种数据类型：

- String
- Hash
- List
- Set
- Sorted Set
- Bitmap
- HyperLogLog

## 命令行

## 客户端

它提供了多种语言的客户端，如：Python，Ruby，PHP，Java，使用方便。
