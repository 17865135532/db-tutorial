说明

本项目的文档按照 gitbook 规范编写，可以直接生成一个 gitbook 电子书站点。

如果想使用 gitbook 电子书，按以下步骤执行命令即可：

# 1. 安装 gitbook-cli
$ npm install -g gitbook-cli
# 2. 安装 gitbook 各种依赖插件
$ gitbook install
# 3. 启动本地的 gitbook 电子书站点
$ gitbook serve

启动成功后，访问 http://localhost:4000 即可看到电子书了。
